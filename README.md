# Shell MFE vs Lib

POC использования Shell (контент обертки) приложения как Remote MFE (ветка main) или как библиотеки (ветка feature/shell-lib)

```html
<vsk-shell>
  <!--ЦЕЛЕВОЙ ПРОДУКТ -->
  <vsk-...></vsk-...>
</vsk-shell>
```

## Запуск приложения

npm:

```sh
npm install
npm run dev
```

yarn:

```sh
yarn
yarn dev
```
